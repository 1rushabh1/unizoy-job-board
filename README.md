# Unizoy Job Board

Full Stack MERN Job Board Application

## Features

- Admin login authentication
- Post / update / delete jobs
- Candidate job applications
- Resume submission
- Admin dashboard
- Modern Tailwind UI

## Tech Stack

Frontend:

- React
- Tailwind CSS
- Axios

Backend:

- Node.js
- Express.js
- MongoDB
- JWT Authentication

## Run Project

Backend:

npm install
node server.js

Frontend:

npm install
npm run dev
